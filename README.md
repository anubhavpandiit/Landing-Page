# Landing-Page
Step 1: Structure (HTML)

First, I created the basic structure of the website using HTML. I added a Header as the main entrance, a Hero Section that users see first, Service Cards to display features, and a Contact Form where users can send messages. 

Step 2: Design (CSS)

Next, I used CSS to style the page. With Flexbox, I organized elements like cards and menus in a clean layout. I also added colors and fonts to give the website a professional look.

Step 3: Responsiveness

To ensure the website works on all devices, I used Media Queries. This allows the layout to adjust automatically so the page looks good on both desktop and mobile screens.

Step 4: Functionality (JavaScript)

Finally, I included JavaScript for interactivity. For example, I created a mobile menu toggle that shows or hides the menu when the user clicks the button.
